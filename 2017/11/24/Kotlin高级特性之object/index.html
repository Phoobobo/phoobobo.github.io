其他使用的注意事项

  • 匿名类作为方法返回值
    匿名类一般只作为本地或者私有声明，如果是公开的且没有继承的，它可能就是作为Any返回，看例子：

  • 访问外部变量不需要像Java一样严格为final

object declaration与expressions的区别

  • object expressions是立即生效的，而object declaration是懒加载的，只有在使用的时候才生效
  • companion object的实例化是依赖于它所在的类的加载的
